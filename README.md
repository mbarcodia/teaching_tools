# teaching_tools
## ML Coding Practice — PyTorch for Weather & Climate                                                                                         
                                                                      
  A small collection of hands-on notebooks for learning and practicing machine learning                                                        
  coding through practical atmospheric science applications.                                                                                   
                                                                                                                                               
  ## Notebooks                                                                                                                                 
                                                                                                                                               
  - **pytorch_weather_tutorial.ipynb** — Introduction to PyTorch tensors, autograd, and                                                        
    neural networks, framed around real weather data. Covers tensor basics, GPU operations,
    fitting the Clausius-Clapeyron equation with linear regression and a small MLP, and a                                                      
    NumPy-to-PyTorch cheatsheet.                                                                                                               
                                                                                                                                               
  - **pytorch_vs_numpy.ipynb** — Side-by-side comparison of PyTorch and NumPy for common                                                       
    array operations, with notes on where they differ.
                                                                                                                                               
  ## Prerequisites
                                                                                                                                               
  - Python, NumPy, basic familiarity with linear regression                                                                                    
  - Notebooks are designed to run on Google Colab (GPU optional but recommended)
                                                                                                                                               
  ## Goals        
                                                                                                                                               
  These notebooks are for learning and practicing ML coding in a practical context —                                                           
  not production code. The focus is on building intuition for how PyTorch works and
  how it applies to real scientific problems.
